## PASOS PARA ACTUALIZAR

### 1. Reemplazar index.html
Copiar el nuevo contenido entregado por ChatGPT y sobrescribir el archivo actual.

### 2. Activar modo oscuro
Cambiar esta línea:

```html
<link rel="stylesheet" href="styles.css">
```

Por:

```html
<link rel="stylesheet" href="styles-dark.css">
```

El conector actual no soporta update directo, por eso este paso es manual.
